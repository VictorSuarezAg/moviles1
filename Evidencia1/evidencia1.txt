1.Que diferencia hay entre un igual ( = ) y un doble ( == )

Un "=" sirve para igualar una variable y un "==" sirve para comparar si el la variable 
o el dato son iguales "x" valor.

2. Cuales son los tipos de datos básico de java
int: representa un numero entero.
float: representa un numero decimal.
double: representa un numero decimal.
boolean: representa un valor entre 0 y 1, se utiliza para validar si algo es verdadero o falso.
char: representa un caracter.
